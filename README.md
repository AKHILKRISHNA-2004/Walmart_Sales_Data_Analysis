# 🛒 Walmart Sales Data Analysis

## 📌 Project Overview
This project focuses on analyzing Walmart’s historical sales data to uncover patterns, trends, and business insights. The analysis emphasizes data cleaning, exploratory data analysis (EDA), and visualization to support data-driven decision-making.

---

## 📊 Dataset
The dataset contains weekly sales information across multiple Walmart stores along with economic and environmental factors:

- Store
- Date
- Weekly_Sales
- Holiday_Flag
- Temperature
- Fuel_Price
- CPI
- Unemployment

Dataset Source: Public Retail Dataset

---

## 🧹 Data Preparation
Key preprocessing steps:

✔ Converted date fields into structured time components (Year, Month, Day, Season)  
✔ Checked and validated missing values  
✔ Identified and handled duplicate records  
✔ Removed incomplete yearly data to avoid analytical bias  

---

## 🔍 Exploratory Data Analysis
The analysis explored:

- Distribution of weekly sales
- Store-wise performance comparison
- Monthly and seasonal sales trends
- Holiday vs non-holiday sales impact
- Correlation between sales and economic indicators

---

## 📈 Visualizations
Multiple visualization techniques were used:

- Histograms for sales distribution
- Bar charts for store & monthly analysis
- Trend lines for time-based sales patterns
- Heatmaps for correlation analysis
- Treemaps & radar charts for multidimensional insights

Tools Used:
- Seaborn
- Matplotlib
- Plotly

---

## 💡 Key Insights
✔ Larger stores tend to generate higher weekly sales  
✔ Holiday weeks show noticeable sales spikes  
✔ Strong correlations observed between Fuel Price & Year  
✔ CPI and Unemployment exhibit meaningful economic relationships
